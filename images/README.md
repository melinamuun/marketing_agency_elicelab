# 섹션 이미지 가이드

랜딩페이지 섹션에 사용할 업그레이드 이미지 7장을 **이 `images/` 폴더**에
아래 **정확한 파일명**으로 저장하세요. (소문자, 공백 없이)

| # | 파일명 | 해당 섹션 |
|---|--------|-----------|
| 1 | `contact.png` | 무료 상담 신청 (폼 + 전화/카톡/이메일) ※ 폼 작동 주의 |
| 2 | `services.png` | 핵심 서비스 (매출로 이어지는 다섯 가지) |
| 3 | `why.png` | 선택의 이유 (작은 가게를 가장 잘 아는 파트너) |
| 4 | `process.png` | 진행 프로세스 (무료 진단부터 확장까지, 6단계) |
| 5 | `pricing.png` | 요금 패키지 (우리 가게 단계에 맞는 패키지) |
| 6 | `results.png` | 성과 사례 (실제 성과로 증명) |
| 7 | `team.png` | 팀 소개 (마케팅 전문가들) |

## 권장 사항
- 형식: `.png` 또는 `.jpg` (사진/일러스트는 `.jpg`가 용량 작음)
- 가로 폭: **1600px 내외** 권장 (너무 크면 로딩 느림)
- 파일당 용량: 가능하면 **300KB 이하**로 최적화 (https://squoosh.app 등)

## 저장 후
파일을 모두 넣으셨으면 "이미지 넣었어요"라고 알려주세요.
제가 `index.html`에서 각 섹션이 이 이미지를 표시하도록 연결하겠습니다.
