# MS마켓 브랜드 가이드라인

> **Brand Guidelines v1.0**
> 일관된 브랜드 경험을 위한 시각·언어 표준 문서

본 가이드라인은 MS마켓의 모든 콘텐츠·광고·자료 제작 시 준수해야 할 기준을 정의합니다.
내부 구성원, 외주 디자이너, 협력사 모두 본 문서를 기준으로 작업합니다.

---

## 1. 브랜드 코어

| 항목 | 정의 |
|------|------|
| 브랜드명 | MS마켓 (MS Market) |
| 의미 | **M**ake **S**ales — "매출을 만든다" |
| 한 줄 정의 | 소상공인의 매출을 키우는 실전형 마케팅 파트너 |
| 비전 | 모든 소상공인이 대기업 수준의 마케팅을 누리는 시장 |
| 미션 | 검증된 시스템으로 마케팅을 표준화해 사장님의 매출을 올린다 |

### 브랜드 퍼스널리티
MS마켓은 **신뢰할 수 있는 든든한 동업자**입니다.

| 키워드 | 의미 |
|--------|------|
| 신뢰 (Reliable) | 투명한 데이터와 약속을 지키는 태도 |
| 실전 (Practical) | 이론이 아닌 매출로 증명하는 실행력 |
| 활기 (Energetic) | 성장을 향한 긍정적이고 적극적인 에너지 |
| 친근 (Friendly) | 사장님 눈높이에서 쉽게 소통 |

## 2. 브랜드 보이스 & 메시지

### 슬로건
> **"광고비는 줄이고, 매출은 올리고."**

### 톤 앤 매너
- 전문가의 신뢰감 + 동네 사장님과 대화하는 친근함의 균형
- 어려운 마케팅 용어는 풀어서 설명 (전문용어 남발 금지)
- 단정적·과장 광고("무조건", "100% 보장") 지양 → 근거 기반 표현

### 표현 가이드

| 권장 (Do) | 지양 (Don't) |
|-----------|--------------|
| "방문자가 ○○% 늘었습니다" | "대박 효과 보장!" |
| "데이터로 확인해 보세요" | "믿고 맡기세요" |
| "사장님의 시간을 아껴드립니다" | "업계 최고의 솔루션" |
| "함께 키워가겠습니다" | "저희만 믿으면 됩니다" |

### 핵심 메시지 (3종)
1. 매출 중심 — 노출이 아니라 매출로 증명합니다.
2. 투명함 — 광고비와 성과를 숨기지 않습니다.
3. 부담 없음 — 무료 진단, 월 단위 유연 계약.

## 3. 로고

### 로고 구성
- **시그니처(국문)** | `MS마켓`
- **시그니처(영문)** | `MS Market`
- **심볼** | 상승 그래프 형태의 'M' (성장·매출 상승 상징) `[심볼 파일 첨부]`

### 최소 여백 (Clear Space)
로고 주변에는 심볼 높이의 **1/2 이상** 여백을 확보합니다.
다른 그래픽·텍스트가 이 영역을 침범할 수 없습니다.

### 최소 사이즈
- 디지털: 가로 **80px** 이상
- 인쇄물: 가로 **20mm** 이상

### 로고 오용 금지 (Misuse)

| 금지 사항 | 설명 |
|-----------|------|
| 비율 왜곡 | 가로·세로 임의 늘림/줄임 금지 |
| 색상 변경 | 지정 컬러 외 임의 색상 적용 금지 |
| 회전·기울임 | 로고 회전 및 입체 효과 금지 |
| 저대비 배경 | 가독성 떨어지는 배경 위 배치 금지 |
| 효과 추가 | 그림자·외곽선·그라데이션 임의 추가 금지 |

## 5. 타이포그래피

### 국문 폰트

| 용도 | 폰트 | 비고 |
|------|------|------|
| 헤드라인 | **G마켓 산스 (Gmarket Sans)** Bold | 임팩트·주목도 |
| 본문 | **Pretendard** Regular / Medium | 가독성·범용성 |

### 영문·숫자 폰트
- **Pretendard** 또는 **Inter** (Pretendard로 통일 가능)

### 위계(Hierarchy) 예시

| 단계 | 크기(웹 기준) | 두께 |
|------|---------------|------|
| H1 (대제목) | 32–40px | Bold |
| H2 (소제목) | 24–28px | Bold/SemiBold |
| 본문 | 16px | Regular |
| 캡션 | 13–14px | Regular |

> 무료·상업적 이용 가능 폰트만 사용해 라이선스 리스크를 제거합니다.

## 6. 그래픽 요소

### 아이콘
- **라인(Outline) 스타일** 통일, 굵기 2px
- 모서리는 약간 둥글게(rounded) 처리해 친근함 유지

### 이미지 톤
- 실제 매장·사장님·고객이 담긴 **현장감 있는 사진** 우선
- 밝고 따뜻한 톤, 과한 필터·왜곡 지양
- 성과 강조 시 그래프·수치 그래픽과 결합

### 그래픽 모티프
- 우상향 그래프 / 화살표 → "성장·매출 상승" 상징
- 둥근 말풍선 → "소통·상담" 상징

## 7. 적용 예시 (Applications)

| 인스타그램 피드 | 네이비 배경 + 오렌지 포인트, 좌측 정렬 헤드라인 |

## 8. 빠른 체크리스트

콘텐츠 발행 전 다음을 확인합니다.

- [ ] 로고 여백·비율·색상이 규정에 맞는가
- [ ] 메인 컬러 비율(네이비 중심)이 지켜졌는가
- [ ] CTA에 오렌지가 적용되었는가
- [ ] 헤드라인=G마켓 산스, 본문=Pretendard인가
- [ ] 과장·단정 표현 없이 근거 기반인가
- [ ] 슬로건/핵심 메시지 톤이 유지되는가
