## 2. 우리가 해결하는 문제

소상공인·중소기업이 마케팅에서 겪는 현실적인 어려움에 집중합니다.

| 사장님의 고민 | MS마켓의 해결 |
|---------------|----------------|
| "광고비만 쓰고 효과를 모르겠어요" | 매출 기준 성과 측정·리포팅 |
| "직접 할 시간이 없어요" | 콘텐츠·광고 운영 풀대행 |
| "대행사에 맡겼다가 데인 적 있어요" | 월 단위 계약·투명 리포트·전담 매니저 |
| "어디서부터 시작할지 모르겠어요" | 무료 진단 후 단계별 로드맵 제시 |
| "예산이 빠듯해요" | 소상공인 맞춤 소액·성과형 패키지 |

## 3. 핵심 서비스

### 3.1 로컬·플레이스 마케팅
- 네이버 플레이스 / 지도 상위노출 최적화
- 방문자 리뷰·예약·톡톡 관리
- 지역 키워드 블로그·체험단 운영

### 3.2 콘텐츠 마케팅
- 인스타그램·릴스·숏폼 기획 및 제작
- 상세페이지·브랜드 스토리 카피라이팅
- 네이버 블로그 / 스마트스토어 콘텐츠 운영

### 3.3 퍼포먼스 광고
- 메타(인스타·페이스북) / 구글 / 네이버 광고 운영
- 타겟 세팅·소재 A/B 테스트·예산 최적화
- 전환 추적(픽셀·전환 API) 세팅

### 3.4 브랜딩·기획
- 로고·BI, 브랜드 톤앤매너 정립
- 신규 오픈·리뉴얼 마케팅 패키지
- 캠페인·프로모션 기획

### 3.5 데이터·자동화
- GA4·광고 데이터 통합 대시보드 구축
- 월간 성과 리포트 자동화
- 고객 DB·재방문 유도(문자·이메일) 시스템

## 4. MS마켓을 선택해야 하는 이유

| 차별점 | 설명 |
|--------|------|
| **소상공인 전문** | 대형 광고주가 아닌, 작은 가게의 구조와 예산에 맞춘 운영 |
| **매출 중심 KPI** | 노출·좋아요가 아니라 문의·예약·매출로 성과를 증명 |
| **투명한 리포트** | 광고비·성과를 월 단위로 공개, 숨김 마진 없음 |
| **전담 매니저** | 한 명의 담당자가 처음부터 끝까지 책임 |
| **부담 없는 계약** | 장기 약정 강요 없이 월 단위 유연 계약 |

## 5. 진행 프로세스

```
① 무료 진단      ② 전략 제안       ③ 계약·온보딩
   매장·온라인         맞춤 채널·예산        목표·KPI 합의
   현황 분석           로드맵 설계          채널 세팅
        │                  │                  │
        ▼                  ▼                  ▼
④ 실행·운영      ⑤ 리포트·개선     ⑥ 확장
   콘텐츠·광고          월간 성과 공유       성과 채널 증액
   집행                 데이터 기반 개선     신규 채널 확대
```

1. **무료 진단** — 매장·SNS·온라인 현황을 분석하고 개선점을 도출합니다.
2. **전략 제안** — 업종·예산에 맞는 채널과 실행 로드맵을 제시합니다.
3. **계약·온보딩** — 목표(KPI)를 합의하고 채널을 세팅합니다.
4. **실행·운영** — 콘텐츠 제작과 광고 집행을 전담 운영합니다.
5. **리포트·개선** — 월간 성과를 투명하게 공유하고 데이터로 개선합니다.
6. **확장** — 성과가 검증된 채널을 키우고 새 채널로 확장합니다.
